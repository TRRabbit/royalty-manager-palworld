# RCON console

RCON is the standard way to send **admin commands** to a game server remotely.
Royalty Manager gives you a friendly console so you never have to memorise
command syntax.

## Opening the console

Open a server and go to the **RCON Console** tab. It has two sub-tabs:

- **Commands** — every admin action as a labelled button.
- **History** — your last commands and the server's replies.

## Built-in command buttons

The common Palworld admin commands are one click away:

| Command | What it does |
|---------|--------------|
| **Announce** | Broadcast a message to everyone on the server |
| **Kick** | Remove a player |
| **Ban** / **Unban** | Block / unblock a player |
| **Save** | Force the world to save now |
| **Shutdown** | Stop the server after a delay, with a message to players |

Commands that are destructive (like Shutdown / DoExit) ask for **confirmation**
first, so you can't trigger them by accident.

## Free command field

Need something specific? Type any raw RCON command in the free field and send
it. Use the **Up / Down arrows** to scroll through your last 10 commands.

## Teleport buttons

From the players table you also get **Teleport to** and **Teleport here**
buttons per player, for quick moderation.

## Help panel

A collapsible **help panel** lists the documented Palworld commands with a short
explanation of each — handy when you're learning.

## PalDefender commands

If you have the [PalDefender mod](mods.md) installed, an extra sub-tab exposes
its **50+ commands** grouped by category (Admin, Whitelist, Players, World,
Items, Pals, and more), with typed inputs and a clear marker on destructive
ones.
