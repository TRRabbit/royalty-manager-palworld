# Mods (UE4SS & PalDefender)

Royalty Manager makes it easy to install the two most useful Palworld server
mods, and to keep them up to date.

Open a server and go to the **Mods** tab.

## UE4SS

**UE4SS** is the modding framework many Palworld mods rely on.

- **Install** — one click downloads the correct UE4SS release from GitHub and
  extracts it into the right place in your server.
- **Check for update** — compares your installed version with the latest release
  and reinstalls if a newer one exists.
- **Open folder** — opens the mods folder in Windows Explorer.

> The app pins a known-good UE4SS version per profile and forces the
> "console off" setting required for a dedicated server, so installing UE4SS
> won't crash your server.

## PalDefender

**PalDefender** is a powerful admin/anti-cheat mod. With it you unlock:

- A **whitelist** that's enforced by the mod (see
  [Players & whitelist](players-and-whitelist.md)).
- **50+ extra admin commands** in the [RCON console](rcon-console.md).
- Richer data for the [live map](live-map.md).

In the Mods tab you can **Install**, **Check for update**, and **Open folder**
for PalDefender, exactly like UE4SS. An access token is generated automatically
on install.

## Other mods (.zip)

You can install other Palworld mods that come as a `.zip`:

- Use the built-in **mod installer** to extract a `.zip` into the correct folder.
- The **Nexus Mods** buttons take you straight to the Palworld category, sorted
  by most recent.

## Performance tools

The Mods/Tools area also offers one-click helpers:

- Add **Windows Defender exclusions** for the server folder (fewer scan-related
  stutters).
- Switch Windows to the **Ultimate Performance** power plan.
- Auto-detect your CPU core count for better threading defaults.
