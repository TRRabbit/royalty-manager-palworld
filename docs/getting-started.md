# Getting started

This page walks you through the very first time you open Royalty Manager.

## The welcome wizard

On first launch, a 3-screen wizard helps you get ready:

1. **Welcome** — a short introduction.
2. **System check** — the app checks that your PC has enough RAM and disk space,
   and that the main Palworld ports look available. Green means you're good;
   a warning tells you what might need attention.
3. **Get started** — create your first server now, or skip to the main window.

You can **Skip** the wizard at any time if you already know what you're doing.

## The main window

After the wizard, you land on the main window. Here's how it's laid out:

- **Top bar** — shows the app version, your edition badge (**Free** / **VIP** /
  **DEV**), and the connection status.
- **Left sidebar** — your navigation. When no server exists yet, you'll see an
  empty state inviting you to **+ Add a profile**.
- **Main area** — shows your server cards, or the screen you've selected.

> A **profile** is one server. Each profile has its own folder, its own
> settings, its own backups, and its own ports. Creating two profiles gives you
> two completely separate servers. (Running more than one server at a time is a
> [VIP feature](free-vs-vip.md).)
