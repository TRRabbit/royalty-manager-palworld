# Live map

The live map shows where players are on the Palworld world map, in real time.

## Opening the map

Open a server and use **Open Live Map**. The map opens in your web browser,
served locally by the app.

> The live map is a [VIP feature](free-vs-vip.md).

## What you see

- The **Palworld world map** with a marker for each connected player.
- A side panel listing players; click a name to focus their location.
- The view **auto-refreshes** every few seconds.

The player positions come straight from your running server, so the map reflects
what's actually happening in game.

## Richer data with PalDefender

If you install the [PalDefender mod](mods.md), the map can be **enriched** with
extra information such as guild bases. Without PalDefender, the map falls back to
the basic player-position view automatically — nothing to configure.

## When no server is running

If no server is online, the map shows a **demo view** so you can see what it
looks like before going live.
