# Free vs VIP

Royalty Manager Palworld is **free to use** for hosting a single server with all
the essentials. **VIP** unlocks the premium features for power users and
communities.

## What's in each edition

| Feature | Free | VIP |
|---------|:----:|:---:|
| Install & run a Palworld server | ✅ | ✅ |
| Full settings editor | ✅ | ✅ |
| Backups (manual) | ✅ | ✅ |
| RCON console | ✅ | ✅ |
| Players: kick / ban / whitelist | ✅ | ✅ |
| Mods (UE4SS / PalDefender) | ✅ | ✅ |
| **Run several servers at once** | — | ✅ |
| **Scheduler** (auto restart/backup/update) | — | ✅ |
| **Discord** (alerts + bot + voice status) | preview only | ✅ |
| **Live map** | — | ✅ |
| **Event mode** | — | ✅ |
| **Import / export setups** | — | ✅ |

> In the free edition, premium screens like **Discord** are shown as a
> **disabled preview** with a **Go VIP** button, so you can see what you'd get.
> You can always create extra server profiles for free — only **running more
> than one at the same time** needs VIP.

## Your edition badge

The top bar shows your current edition:

- **Free** — the free edition.
- **VIP** — a valid licence is active.
- **DEV** — developer/testing mode (not something customers see).

## How to activate VIP

1. Get a licence key from the official website / store.
2. Open **Settings**, find the **Licence** section.
3. Paste your key and activate.

The premium features unlock immediately and the badge switches to **VIP**.

## Billing & trial

- Licences are handled through **Lemon Squeezy** (which also handles worldwide
  VAT/tax as the merchant of record).
- A **14-day free trial** is available, no credit card required.
- The app re-checks your licence periodically. If a subscription is cancelled,
  the app gracefully returns to the free edition (worst case after a short
  offline grace period) — your data is never lost.

## Reset application

Settings also has a **Reset application** button (in the danger zone). It wipes
the app's own data and starts fresh. It does **not** delete the Palworld server
files or worlds on your disk.
