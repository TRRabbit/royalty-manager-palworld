# Event mode

Event mode lets you run a temporary event — like a double-XP weekend — and have
the server **automatically return to normal** when it ends. No need to remember
to switch the settings back.

> Event mode is a [VIP feature](free-vs-vip.md).

## How it works

1. You start an event (for example, *Double XP for 48 hours*).
2. The app **snapshots** your current settings, then applies the event's
   overrides and restarts the server.
3. When the event's end time arrives, the app **restores** your original
   settings automatically and restarts again.
4. If Discord is set up, it announces the start and the end.

Because the original settings are snapshotted, you always get back exactly what
you had before — even if the app was closed and reopened during the event.

## Built-in presets

Ready-to-use events:

- **Double XP**
- **Double Drops**
- **Halloween**
- **Christmas**
- **Anniversary**

## Custom events

Create your own with:

- A **name** and **description**
- A **duration**
- Multipliers for **XP**, **drops**, and **capture rate**

Custom presets are saved locally so you can reuse them.
