# Installation

## 1. Download

Go to the **[latest release](https://github.com/TRRabbit/royalty-manager-palworld/releases/latest)**
and download the installer (a `.exe` file, inside a `.zip`).

Unzip it if needed, then double-click the installer.

## 2. The "Windows protected your PC" warning

The first time you run the installer, Windows SmartScreen may show a blue
window that says **"Windows protected your PC"** and *"unknown publisher"*.

**This is normal for new independent software** and does **not** mean the file
is dangerous. It simply means the app is new and hasn't built up a reputation
with Microsoft yet.

To continue:

1. Click **More info**.
2. Click **Run anyway**.

> Prefer not to? You can scan the file with your antivirus or
> [VirusTotal](https://www.virustotal.com) first. You can also ask in our
> [Discord](https://discord.gg/RjzkFaJsC5) if you have any doubt.

## 3. The setup wizard

The installer is a normal Windows wizard. It lets you choose:

- **Where to install** the app (a default folder is suggested).
- **Desktop shortcut** — create an icon on your desktop (recommended).
- **Start Menu shortcut** — added automatically.
- **Launch at Windows startup** — optional, off by default. Turn this on if you
  want your server PC to be ready as soon as Windows boots.

Click through the wizard, then **Finish**. The app can launch automatically at
the end.

> **Taskbar pinning:** Windows does not allow apps to pin themselves to the
> taskbar. If you want it there, right-click the running app on the taskbar and
> choose **Pin to taskbar**.

## 4. First launch

When the app opens for the first time, a short **welcome wizard** appears. See
[Getting started](getting-started.md) for what happens next.

## Uninstalling

Royalty Manager installs like any normal Windows app. To remove it:

- Open **Settings → Apps → Installed apps**, find **Royalty Manager**, and
  choose **Uninstall**, **or**
- Use the **Uninstall** shortcut in the Start Menu folder.

Uninstalling removes the app. It does **not** delete the Palworld server files
or worlds you installed — those stay on your disk so you don't lose your save.
