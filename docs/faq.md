# FAQ

Short answers to common questions. See [Troubleshooting](troubleshooting.md) for
fixes to specific problems.

### Is it free?

Yes — you can host a single Palworld server with all the essentials for free.
Premium features (several servers at once, Discord, scheduler, live map, event
mode, import/export) are **VIP**. See [Free vs VIP](free-vs-vip.md).

### Do I need a Steam account or to install Steam?

No. The app uses SteamCMD (Steam's official tool) under the hood and downloads
it for you. You don't need a Steam account.

### Do I need to know how to use a command line or edit config files?

No. That's the whole point — everything is done through the app's interface.

### Will this work on my PC?

If you're on **Windows 10 or 11 (64-bit)** with enough RAM, yes. Hosting a
server needs a fair amount of memory — see [System requirements](system-requirements.md).

### Can I host and play on the same PC?

Yes, but you need enough RAM for **both** the server and the game at once. 16 GB
or more is recommended for that.

### Why does Windows say "unknown publisher"?

Because the app is new independent software. It's safe to run — click
**More info → Run anyway**. See [Installation](installation.md).

### Can my friends join over the internet?

Yes, but you'll usually need to **forward ports** on your router (8211 and 27015
UDP). See [Creating your first server](creating-your-first-server.md#step-5--let-friends-join)
and [Troubleshooting](troubleshooting.md#players-cant-join).

### Can I run more than one server?

You can **create** as many server profiles as you like for free. **Running more
than one at the same time** is a [VIP feature](free-vs-vip.md).

### Will I lose my world if I uninstall or reset the app?

No. Uninstalling the app and the **Reset application** button do **not** delete
your Palworld server files or worlds. They stay in the folder you chose. Still,
keep [backups](backups.md).

### How do updates work?

The app updates itself automatically when a new version is released. See
[Updating the app](updating.md).

### Does it support mods?

Yes — UE4SS and PalDefender install in one click, and you can add other `.zip`
mods too. See [Mods](mods.md).

### What languages are supported?

The interface is available in **English** and **French**, switchable in Settings.
